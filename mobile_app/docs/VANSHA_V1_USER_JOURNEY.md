
# Vansha V1 User Journey

## User Types

### 1. Stranger User

Downloads Vansha from App Store.

No invite.

No existing family connection.

Creates:

- Self Identity
- Parent Identities
- Sibling Identities
- Spouse Identity (optional)

Result:

Personal Family Graph created.

---

### 2. Invited User

Receives invite from existing member.

Claims existing identity.

Result:

Identity becomes verified.

Graph expands automatically.

---

### 3. Marriage Bridge User

Existing user connects to spouse from another family circle.

Result:

Two family circles become linked.

Relationship Intelligence expands across both families.

---

## Core User Journey

1. Sign Up
2. Verify Mobile
3. Choose Language
4. Add Parents
5. Add Siblings
6. Add Spouse
7. View Family Graph
8. Tap Person
9. View Relationship Intelligence
10. Invite Relative

---

## V1 Success Criteria

User can answer:

- Who is this person to me?
- What do I call them?
- How is my child related to them?
- How is my spouse related to them?
