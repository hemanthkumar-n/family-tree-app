# MVP Scope

## Included in V1
- User authentication
- Self-first onboarding
- Add parents, siblings, spouse, children, grandparents
- Interactive family tree visualization
- Person profile management
- Invite family members
- Profile claim flow
- Relationship explainer

## Excluded from V1
- Chat/messaging
- Social family feed
- AI storytelling
- Document vault
- Photo scanning
- Voice memories
